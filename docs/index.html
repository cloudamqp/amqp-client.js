properties (object) publish properties
Name Description
properties.contentType string mime type, eg. application/json
properties.contentEncoding string eg. gzip

properties.deliveryMode number 1 for transient, 2 for persisent

properties.messageId string

properties.type string
properties.userId string
properties.appId string
mandatory (boolean?) if the message should be returned if there's no queue to be delivered to
immediate (boolean?) if the message should be returned if it can't be delivered to a consumer immediately (not supported in RabbitMQ)
Returns
Promise<number, AMQPError>: fulfilled when the message is enqueue on the socket, or if publish confirm is enabled when the message is confirmed by the server
▸ basicQos(prefetchCount, prefetchSize, global)

Set prefetch limit. Recommended to set as each unacknowledge message will be store in memory of the client. The server won't deliver more messages than the limit until messages are acknowledged.

basicQos(prefetchCount: number, prefetchSize: number, global: boolean)
Parameters
prefetchCount (number) number of messages to limit to
prefetchSize (number = 0) number of bytes to limit to (not supported by RabbitMQ)
global (boolean = false) if the prefetch is limited to the channel, or if false to each consumer

params (object = {})
Name Description

params.durable boolean (default name!=="") if the queue should survive server restarts
